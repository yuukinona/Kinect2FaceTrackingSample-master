FaceTrackingSamples
===================

Kinect v2のFaceTrackingサンプル集です。

# サンプル一覧
* FaceTrackingBasic
    + 顔を検知して表情を分析しリアルタイムに表示します
    + 顔のパーツの座標(右目、左目、花、口の左端右端)を特定し、カラー情報と合わせて表示します
* Face3DModelTracking
    + 顔を詳細に分析します
    + 顔の頂点、TriangleIndicesを特定し3Dモデルを作成します
    + 顔の動きを取得しリアルタイムに表示します
    + ModelBuilderを使用し、顔の全体像の3Dモデルを作成します
    + 未完成
    + 実装予定(途中)な機能
        - HDFacePointを用いて、色情報と同時に顔の頂点にマーカーを置く
